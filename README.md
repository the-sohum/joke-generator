# 😂 Dad Joke Generator

A fun, colorful web app that fetches random dad jokes from a public API — because the world needs more groan-worthy humor.

## 🌐 Live Preview

Open `index.html` in your browser and click **New Joke** to get started!

## ✨ Features

- Fetches random dad jokes from the [icanhazdadjoke.com](https://icanhazdadjoke.com/) public API
- Clean, responsive UI with a bold color scheme
- One-click joke generation

## 🛠️ Tech Stack

- **HTML5**
- **CSS3** (custom styles, Google Fonts – Poppins)
- **Vanilla JavaScript** (Fetch API)

## 📁 Project Structure

```
├── index.html    # Main HTML page
├── styles.css    # Styling and layout
└── script.js     # Joke fetching logic
```

## 🚀 Getting Started

No installation or build step needed — this is a plain HTML/CSS/JS project.

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/dad-joke-generator.git
   ```
2. Open `index.html` in any modern browser.
3. Click **New Joke** and enjoy! 😄

## 🔌 API

Jokes are fetched from the free, open [icanhazdadjoke API](https://icanhazdadjoke.com/):

```
GET https://icanhazdadjoke.com/
Headers: { "Accept": "application/json" }
```

No API key required.
